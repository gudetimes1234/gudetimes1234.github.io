# charliegude.com

Personal site. Hugo, single page, no theme.

## Local dev

```
hugo server
```

Visit http://localhost:1313

## Editing content

- **Work history**: edit `data/roles.yaml`
- **Layout / CSS**: edit `layouts/index.html`
- **Site config**: edit `hugo.toml`

## Deploy

Auto-deploys to Vercel on push to main.
